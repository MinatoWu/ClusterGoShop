# ClusterGoShop
分布式微服务电商项目
GO-Shop 是一个用golang实现的基于微服务架构的分布式电商系统，旨在提供高效、可靠的电商服务。系统整体采用grpc通信并且基于web,srv双层架构，结合相关技术和工具实习核心业务，在高并发场景下具备高可用性和可扩展性。
