# ClusterGoShop
分布式微服务电商系统  

该项目是golang实现的基于微服务架构的分布式电商系统，旨在提供高效、可靠的电商服务。系统整体基于web,srv双层架构并且采用grpc进行服务通信,结合相关中间件构造核心业务在高并发场景下具备高可用性和可扩展性。
